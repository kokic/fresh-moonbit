# [MoonBit] Language Support for [Fresh]

Syntax highlighting and LSP integration for [MoonBit].

## Installation

Use Package: Install from URL in the command palette (`Ctrl+P`) and enter:

```
https://github.com/kokic/fresh-moonbit
```
